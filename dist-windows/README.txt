PENALTY SHOOTOUT — DISTRIBUSI WINDOWS
====================================

Isi folder ini:
  - aplikasi_perhitungan_penalty.exe   (binary aplikasi Windows, sudah jadi)
  - run.bat                (klik 2x -> buka Command Prompt -> jalan)
  - README.txt             (file ini)

CARA PAKAI (user awam):
  1. Extract folder ini ke mana saja.
  2. Klik 2x "run.bat".
  3. Command Prompt otomatis terbuka dan permainan jalan.
  4. Setelah selesai, tekan sembarang tombol untuk menutup jendela.

SYARAT:
  - Windows 64-bit (x86_64).
  - TIDAK perlu install apa-apa, TIDAK perlu koneksi internet.
    Semua dependensi (PDCurses) sudah dibungkus ke dalam .exe.

CATATAN:
  - .exe ini untuk Windows 64-bit. Untuk 32-bit butuh rebuild (i686-mingw).
  - App berjalan di Command Prompt (console), bukan window grafis.
