# Plan: Perbaikan UX/UI Aplikasi Penalty Shootout

## Problem Statement

TUI aplikasi penalty shootout saat ini terlalu sederhana:
- Menu utama hanya kotak teks biasa tanpa visual menarik
- Pendaftaran peserta tidak ada progress bar atau indikator visual
- Input tendangan tidak ada panduan zona atau visualisasi skor
- Ranking tidak ada medal/indikator juara
- Search hasilnya plain text
- Recap kurang informatif sebagai dashboard
- Tidak ada splash screen atau branding

## Scope

Hanya modifikasi layer **TUI adapter** dan **Surfaces** (sesuai AES architecture):
- `src/tui/infrastructure_tui_adapter.h` — tambah color pairs & helper functions
- `src/tui/infrastructure_tui_adapter.c` — implementasi helper functions
- `src/cli/surfaces_menu_command.c` — menu utama lebih menarik
- `src/cli/surfaces_registration_command.c` — progress bar & layout lebih baik
- `src/cli/surfaces_scoring_command.c` — zone guide & scoring visual
- `src/cli/surfaces_ranking_command.c` — medal indicators & tabel berbingkai
- `src/cli/surfaces_search_command.c` — hasil lebih terstruktur
- `src/cli/surfaces_recap_command.c` — dashboard summary & zone chart

**TIDAK MODIFIKASI:** taxonomy, contract, capabilities, agent, root, tests, Makefile.

## Changes

### 1. TUI Adapter — Tambah Color Pairs & Helpers

**File: `src/tui/infrastructure_tui_adapter.h`**

Tambah color pair definitions:
```c
#define COLOR_GOLD     7   /* Emas untuk juara 1 */
#define COLOR_SILVER   8   /* Perak untuk juara 2 */
#define COLOR_BRONZE   9   /* Perunggu untuk juara 3 */
#define COLOR_INFO    10   /* Info/netral */
#define COLOR_WARNING 11   /* Peringatan kuning */
#define COLOR_DIM     12   /* Teks redup */
```

Tambah helper function prototypes:
```c
void tui_progress_bar(int row, int col, int width, int current, int total);
void tui_divider(int row, int col, int width);
void tui_status_bar(int row, const char *left, const char *center, const char *right);
void tui_header_box(int row, int col, int width, int height, const char *title);
void tui_medal(int row, int col, int rank);
```

**File: `src/tui/infrastructure_tui_adapter.c`**

Implementasi:
- `tui_progress_bar()` — gambar bar progres ASCII: `[████████░░░░] 5/7`
- `tui_divider()` — garis pemisah horizontal dengan karakter box
- `tui_status_bar()` — baris status di bawah layar dengan 3 kolom (kiri/tengah/kanan)
- `tui_header_box()` — kotak berjudul (box + title di tengah border atas)
- `tui_medal()` — tampilkan medal emoji/simbol untuk peringkat 1/2/3

### 2. Menu Utama — Splash & Status

**File: `src/cli/surfaces_menu_command.c`**

Perubahan:
- Tambah splash/branding header: `"=== PENALTY SHOOTOUT ==="` dengan border dekoratif
- Status bar di bawah: tampilkan jumlah peserta terdaftar & progres
- Menu items pakai simbol navigasi: `↑↓` untuk panah, `[ENTER]` untuk pilih
- Status tags berwarna: [AKTIF] hijau, [BLOKIR] merah, [SELESAI] cyan, [SUDAH] kuning
- Footer: `"↑↓ Navigasi  [Enter] Pilih  [Esc] Keluar"`

### 3. Pendaftaran — Progress & Layout

**File: `src/cli/surfaces_registration_command.c`**

Perubahan:
- Progress bar di atas: `[████░░░░] 2/7 peserta`
- Daftar peserta pakai numbering: `1. Budi Santoso  ✓`
- Error messages di-posisi bawah layar (baris tetap, tidak hardcode row 20)
- Input field dengan border/box
- Konfirmasi selesai: "X peserta terdaftar. Tekan Enter."

### 4. Scoring — Zone Guide & Visual

**File: `src/cli/surfaces_scoring_command.c`**

Perubahan:
- Mini zone reference table di pojok: `Zona: 0=Miss 1=Mudah 2=Sedang 3=Sulit 4=SgtSulit 5=Sempurna`
- Progress bar tendangan: `[███░░░░] 3/7`
- Riwayat tendangan pakai simbol: `Z3 Z2 Z5 Z1` dengan warna (hijau untuk tinggi, merah untuk rendah)
- Hasil tendangan: visual feedback `Zona 5 → ★ 5 poin!` atau `Zona 0 → ✗ 0 poin`
- Skor sementara ditampilkan prominent

### 5. Ranking — Medal & Table

**File: `src/cli/surfaces_ranking_command.c`**

Perubahan:
- Medal indicators: `🥇 1` (gold), `🥈 2` (silver), `🥉 3` (bronze)
- Atau fallback ASCII: `[*] 1`, `[+] 2`, `[-] 3`
- Tabel dengan border box yang proper
- Baris juara 1 highlight gold, juara 2 silver, juara 3 bronze
- Tie indicator: jika seri, tampilkan `=` atau label `SERI`
- Footer ringkasan: total peserta, skor tertinggi, rata-rata

### 6. Search — Structured Result

**File: `src/cli/surfaces_search_command.c`**

Perubahan:
- Hasil pencarian dalam box terstruktur
- Info peserta dalam format label-value: `Nama: Budi Santoso`
- Riwayat tendangan: `Tendangan: [3] [2] [5] [1] [4] [0] [3]` dengan warna
- Zone frequency: bar chart mini `█ ███ ██ █` atau tabel
- "Tidak ditemukan" lebih jelas dengan saran

### 7. Recap — Dashboard

**File: `src/cli/surfaces_recap_command.c`**

Perubahan:
- Summary stats di atas: `Peserta: 6 | Skor Tertinggi: 28 | Rata-rata: 22.3`
- Tabel dengan border yang lebih rapi
- Zone frequency: tampilkan sebagai bar chart mini per peserta
- Highlight juara 1
- Footer dengan total tendangan keseluruhan

## Constraints

- C99 standard
- ncurses only (no external libs)
- AES architecture: hanya modify `infrastructure_tui_adapter` dan `surfaces_*` files
- Tests excluded from TUI/surfaces (Makefile lines 30-37) — tidak ada test yang terpengaruh
- Windows compatibility via `#ifdef _WIN32` sudah ada di adapter
